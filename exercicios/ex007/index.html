Itálico / Ênfase

Nesta frase temos uma palavra em Itálico usando a tag < i > (não semântica).

Texto marcado

Podemos criar um texto marcado usando a tag < mark > e podemos mudar a cor usando css.

E em outro parágrafo se definirmos o style da tag < mark > todas as marcações estaram com a nova cor

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno
obs: a tag < big > está obsoleta enquanto a < small > ainda pode ser usada

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag < del >.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag < ins >.

Existe também o sublinhado com a tag < u > (não semântica).
